# Tool Integration Source Notes

## SpriteFusion Pixel Snapper

Repository: https://github.com/Hugo-Dz/spritefusion-pixel-snapper

The project documents CLI installation through Brew or Cargo, single-image and directory processing, color-count quantization, `--pixel-size` override, custom palettes, and a WASM `process_image` integration.

## Nearest-Neighbour-Upscale

Repository: https://github.com/cole8888/Nearest-Neighbour-Upscale

The project documents a C implementation and example driver for nearest-neighbor enlargement of 24-bit or 32-bit PNG files using an integer scale argument.

Review the upstream repositories and their licenses before shipping or redistributing binaries.
